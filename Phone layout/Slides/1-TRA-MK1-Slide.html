THE OBJECTIVE

Trajan Scientific and Medical required a method to validate the long-term durability and cycle endurance of chromatography syringes. Tasked with developing a functional tester from an outdated, unused chromatography machine, the project involved reverse-engineering a chromatography machine to fully understand its core drive mechanisms.

THE EXECUTION

The system was stripped down and simplified to its foundational components. Utilizing aluminum extrusions, salvaged parts, and custom 3D-printed mounts, a specialized test jig bench was engineered. A Raspberry Pi was integrated and programmed using Python to manage control logic, interface with limit switches for programmable cycle counts, and automated data recording.

THE IMPACT

The fully assembled prototype successfully performed automated syringe cycling, capturing critical failure data while proving the need for a future production ready unit.
